# Setup Requirements

## Project type

SPA application for practicing modern frontend development.

## Language

JavaScript

## Main stack

- React
- React Router
- Redux Toolkit
- RTK Query
- Webpack
- Native CSS

## API

- DummyJSON API

## Code quality

- ESLint
- Prettier
- Husky
- Consistent code style and formatting

## Functional scope

- At least 3 pages
- Routing between pages
- Global state via Redux Toolkit
- Data fetching and caching via RTK Query
- Responsive layout for mobile and desktop

## Project constraints

- No create-react-app
- No UI component libraries for the main interface
- Use modular and readable project structure
- Follow React best practices:
  - functional components
  - hooks
  - separation of logic and UI

## Optional libraries

- Tables / charts / maps may use third-party libraries if needed

## Quality requirements

- Clean code
- Consistent formatting
- Structured commits
- README with:
  - short project description
  - list of dependencies
  - run instructions
  - deployment link

## Delivery

- GitHub repository
- Deployed version on Netlify / Vercel / GitHub Pages
